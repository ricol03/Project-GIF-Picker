# Project GIF Picker
![Placeholder logo](doc/logo.svg)

This GTK app pretends to be a small but effective tool to have a local GIF library easily accessible. It will allow the user to copy the pretended GIF to the clipboard, aswell as to search through the library, to effectively go through large folders.

## Building
The process uses Ninja, Meson and valac, so make sure to have these installed. As for the commands themselves, make sure to have the necessary dependencies installed and build the Meson project as normal.

To install it permanently on your system, run:
```meson install -C <your build directory>```
